# 원문 서고(근대 단편 20편) 출처

- 성격: 옴니버스형이지만 liaozhai/samguk-yusa와 달리 **저자 20편이 모두 다른 단편형 원문 묶음**.
  각색 파이프라인(번역·교정·조판) 대상이 아니라 **저작권 만료 원문을 그대로 신는 "원문 서고"** 콘텐츠용 소스.
- 판본: 한국어 위키문헌(ko.wikisource.org) 수록 현대 표기 전사본
- 저작권 상태: 전원 퍼블릭 도메인 (저자 1962년 이전 사망 → 사후 50년 규정으로 만료, 2013년 개정법 이전 만료 확정)
  - 작가별 몰년: 김유정 1937 · 현진건 1943 · 나도향 1926 · 이효석 1942 · 김동인 1951 · 계용묵 1961 · 채만식 1950 · 최서해 1932
- 정규화 내역: 없음. 위키문헌 원문 그대로(문단 = 빈 줄 구분), 개행 정규화 외 손대지 않음.
- 파일 단위: **단편형 원문** — 원문 그대로 1파일 = 1작품(챕터 분할 없음).
  파일명 `NN-저자-제목.md`, 각 파일 앞에 YAML 프론트매터(`title`/`author`/`year`/`type`/`source`/`source_name`/`license`)로 개별 출처를 자체 기록.
- 비고: 이 중 김유정 5편(동백꽃·봄봄·금 따는 콩밭·만무방·땡볕)은 `chaekbi` 앱의 `content/kim-yujeong-anthology/`(각색 예정, 화 단위 옴니버스)와 제목이 겹치나 **별개 콘텐츠**다.
  하나는 원문 그대로(이 폴더), 하나는 향후 각색본(김유정 단편집)이라 slug·work가 서로 다르다. 혼동 주의.

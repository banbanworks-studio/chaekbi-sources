# 몬테크리스토 백작 원문 출처

- 원저자: Alexandre Dumas (공저: Auguste Maquet) — 원저는 프랑스어
- 원제: Le Comte de Monte-Cristo (Project Gutenberg 표준 텍스트는 **영역본**)
- 판본: **영역본 채택** (확정) — Project Gutenberg #1184 = 1846년 익명 번역(Chapman & Hall), 완역본(unabridged). 프랑스어 원문은 채택하지 않음.
- 소스 경로 (표준): `https://raw.githubusercontent.com/GITenberg/The-Count-of-Monte-Cristo_1184/master/1184.txt` (GITenberg 미러)
- 소스 경로 (예비): Project Gutenberg eBook #1184 (gutenberg.org/ebooks/1184)
- 판본 검증 마커: 검증 완료 (2026-07-06) — 표준 문두("On the 24th of February, 1815, the look-out at Notre-Dame de la Garde...") 및 117장 구성 확인
- 저작권 상태: 퍼블릭 도메인 — 1846년 번역 + 뒤마 1870년 사망
- 정규화 내역: CRLF → LF 정규화, PG 라이선스 보일러플레이트(머리말/꼬리말) 제외 후 본문(Chapter 1~117)만 저장. 원문 텍스트 자체는 무수정.
- 파일 단위: `ch-XXX.txt` (3자리 zero-pad, 총 117개 챕터로 확정)
- 비고 (이본 문제 등 저본 선정 근거, 해당 시): 영역본 채택 사유 — 리텔링 파이프라인 기준 오독 리스크 최소화 + PD 조항 충족. 프랑스어 원문은 역량 검증 후순위 군이라 배제. 역자 미상.
- 각색 시 주의: 이 1846년 번역은 완역이나 빅토리아 시대 완화 지점이 2곳 있음 — ① 외제니의 동성애적 묘사, ② 백작의 해시시 탐닉. 각색 시 인지할 것.

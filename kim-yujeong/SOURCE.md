# 김유정 단편집 원문 출처

- 원저자: 김유정 (金裕貞, 1908~1937)
- 원제: (단편집이므로 작품 전체를 아우르는 원제 없음 — 편별 원제는 각 파일 단위로 관리)
- 판본: 한국어 위키문헌(ko.wikisource.org) 수록본 — 저자 사후 70년 경과로 대한민국 저작권 만료 상태
- 소스 경로 (표준): `https://ko.wikisource.org/wiki/저자:김유정` (작품별 개별 문서로 연결)
- 소스 경로 (예비): 개별 작품 문서는 `https://ko.wikisource.org/wiki/<제목>` 패턴(제목은 URL-encode). 동명이인/동명작품 충돌 시 `<제목> (김유정)` 형태로 소거(예: `가을 (김유정)`, `두꺼비 (김유정)`, `정조 (김유정)`). 실제 확인된 개별 문서 URL은 아래 "파일 목록" 표 참고.
- 판본 검증 마커: 2026-07-07, MediaWiki API(`action=query&prop=revisions&rvslots=main`, ko.wikisource.org)로 아래 30개 표제작 전부를 실제로 조회·대조함. 그중 29편은 개별 문서 원문(위키텍스트)을 직접 받아 tale-XXX.txt로 저장·검증 완료. 나머지 1편("봄밤")은 위키문헌의 `봄밤` 문서가 김유정 작품이 아니라 김소월 『진달래꽃』 수록시 "봄 밤"(`진달래꽃 (시집)/봄 밤`)으로 넘겨주기(redirect)되어 있어 접근 불가 확인(`봄밤 (김유정)` 소거 문서도 없음 — 미생성 문서). 템플릿 처리 규칙(SIC/드러냄표/복원/c/더더(더)크게 등)은 `action=parse&prop=text`로 렌더링 결과를 표본 대조하여 확정. 나머지 세부 대조(다른 판본과의 자구 비교 등)는 [2차 작업 시 확정].
  - 추가 확인 결과: 기존에 "개별 문서 없음(추가 확인 필요)"로 남겨두었던 5편 중 4편은 실제로 개별 문서가 존재함을 확인함 — 홍길동전 (김유정), 두포전, 형, 애기 (모두 `EXISTS`, MediaWiki API로 조회 성공). 이번 작업 범위(위 30편 목록)에는 포함되어 있지 않으므로 tale-XXX.txt로는 아직 만들지 않았음 — 별도 작업으로 편입 여부 결정 필요. "잃어진 보석"만 여전히 문서 없음(`MISSING`) 확인.
- 저작권 상태: 퍼블릭 도메인
- 정규화 내역: 위키텍스트(HTML 아님)를 MediaWiki API로 직접 받아 가공. 제거한 것 — 머리말 헤더 템플릿(제목/저자/이전·다음 등 UI), `{{위키백과|...}}` 상호참조 박스, `{{정리 필요}}` 정비 배너, HTML 주석(`<!-- 줄나눔에 따른 단어분리 -->` 등 편집자 메모), `<ref>...</ref>` 각주(위키 편집자의 서지 주석, 원문 아님), 말미의 `라이선스`/`주석`/`분류` 섹션 및 PD-old 템플릿. 변환한 것 — 위키링크 `[[…|표시]]`/`[[…]]`는 표시 텍스트만 남김, `<br/>`는 줄바꿈으로, `{{문단 그림}}`/`<center>…</center>` 장면전환 장식은 원문 그대로 보존(양쪽 다 동일한 "🙝 🙟" 장식으로 렌더링됨을 확인), 옛한글 특수 템플릿(`{{SIC|A|B}}`→A, `{{드러냄표|A|…}}`→A, `{{복원|A}}`→A, `{{c|…}}`/`{{더더크게|…}}`/`{{더더더크게|…}}`→내부 텍스트)은 실제 렌더링(action=parse) 결과와 대조해 "화면에 표시되는" 파라미터만 남기는 규칙으로 통일. CRLF→LF, 연속 개행 3개 이상→2개로 정리. 본문 문장 자체는 요약·의역·축약 없이 원문 그대로.
- 파일 단위: `tale-XXX.txt` (3자리 zero-pad). 위키문헌 저자 페이지에 나열된 표제작은 실제로는 28편이 아니라 30편이며(원래 이 문서의 "총 28편" 표기는 오기 — 아래 나열된 항목은 30개), 순번은 저자 페이지 게재 순서를 그대로 따름:
  산골 나그네, 총각과 맹꽁이, 소낙비, 금 따는 콩밭, 노다지, 금, 떡, 산골, 만무방, 솥, 봄봄, 아내, 심청, 봄과 따라지, 가을, 두꺼비, 봄밤, 이런 음악회, 동백꽃, 야앵, 옥토끼, 생의 반려, 정조, 슬픈 이야기, 세발자전거, 따라지, 땡볕, 연기, 정분, (번역) 귀여운 소녀
  - `tale-001.txt` ~ `tale-030.txt`로 확정. 다만 17번째 "봄밤"은 원문 문서 자체가 존재하지 않아(위 판본 검증 마커 참고) `tale-017.txt`를 생성하지 않고 결번으로 둠(총 29개 파일 생성).
  - 파일 내 표기 규칙: 첫 줄 = 원제(주석 없이 텍스트만), 둘째 줄 = 빈 줄, 셋째 줄부터 본문. 줄바꿈은 LF.
  - 개별 문서 없이 목록에만 있는 작품(2026-07-07 재확인): 잃어진 보석(문서 없음, MISSING 확인). ※ 홍길동전(김유정), 두포전, 형, 애기는 개별 문서가 실제로 존재함이 이번에 확인되었으나 이번 30편 목록에는 포함되지 않아 파일화하지 않음(위 판본 검증 마커 참고).
- 비고 (이본 문제 등 저본 선정 근거, 해당 시): 수록 목록·수록 순서 확정, 그리고 "홍길동전(김유정)"이 [honggildong/](../honggildong/) 폴더와 별개로 이 폴더에 속하는 것이 맞는지(원작 번안이라 김유정 항목이 맞음) 허브 확인 권장.

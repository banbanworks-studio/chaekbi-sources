# 삼국유사 원문 출처

- 원저자: 일연 (一然)
- 원제: 三國遺事 (삼국유사) — 한문(漢文) 원문
- 판본: **한문 원문 채택** (확정) — 대조 기준은 中宗壬申本(중종임신본, = 규장각본, 1512년 간행)
  - 국사편찬위원회 한국사데이터베이스 "한국 고대 사료 DB"가 권1~5 전권에 대해 규장각본(중종임신본)을 기본으로 제공, 대조용으로 활용
- 소스 경로 (표준): `https://zh.wikisource.org/wiki/三國遺事/卷第一` ~ `卷第五` (한자 숫자 一二三四五 사용, 확인 완료). 문서 구조:
  - `卷第一`: 王曆第一 + 紀異第一
  - `卷第二`: 紀異第二(下)
  - `卷第三`: 興法第三 + 塔像第四
  - `卷第四`: 義解第五
  - `卷第五`: 神咒第六 + 感通第七 + 避隱第八 + 孝善第九 + 跋(1512년 중종임신본 간행 발문)
  - 총목차는 `https://zh.wikisource.org/wiki/三國遺事` (메인 문서)에 있음. 저본 스캔은 `三國遺事 卷第一 1512年 奎章閣本.pdf` 등 5개 Index: 파일(奎章閣本, 규장각본과 일치)
  - **주의**: `卷第一`은 위키문헌 mainspace의 raw wikitext만으로는 王曆第一만 노출되고 紀異第一(古朝鮮 등 주요 설화 전체)이 보이지 않음 — 이는 ProofreadPage `<pages index=... from="1" to="103" />` 트랜스클루전으로 Page: 네임스페이스(개별 교정 완료 스캔 페이지)에서 동적으로 불러오는 구조이기 때문. 실제 렌더링(브라우저에서 보는 화면)에는 紀異第一이 정상적으로 포함되어 있음을 확인. 이번 작업에서는 Page:三國遺事 卷第一 1512年 奎章閣本.pdf/31~103 (紀異 시작 지점인 31쪽부터 마지막 103쪽까지, 73개 페이지)을 MediaWiki API로 직접 가져와 복원함. 卷第二~卷第五는 이런 트랜스클루전 구조 없이 mainspace wikitext 자체에 전체 본문이 들어있어 별도 조치 불필요.
- 소스 경로 (예비): `https://db.history.go.kr/ancient/level.do?itemId=sy` (한국사데이터베이스, 규장각본/중종임신본 대조용 — 권별 정확한 URL 패턴은 재확인 필요)
- 판본 검증 마커:
  - 卷第一 紀異第一 첫 항목 "古朝鮮〈王儉朝鮮〉" 및 단군 신화 본문("《魏書》云：乃往二千載有壇君王儉...") 확인 완료.
  - 卷第一 紀異第一 마지막 항목이 ToC상 "長春郎　罷郎"인데, 실제 위키문헌 표기는 "春郎 罷郎"으로 "長" 자가 앞 문단 끝("...太宗之號長。")에 붙어 있는 오식(誤植)이 있음 — 위키문헌 자체의 표기이므로 임의로 옮기지 않고 원문 그대로 보존함(2행에 걸친 원문 그대로).
  - 卷第二 첫 항목 "文虎王法敏"(=文武王, 避諱字 표기) 확인, 마지막 항목 "駕洛國記"(居登王~仇衡王 世系 포함) 확인. 목차 전 항목과 대조하여 누락 없음 확인.
  - 卷第三(興法第三/塔像第四), 卷第四(義解第五), 卷第五(神咒第六/感通第七/避隱第八/孝善第九 + 跋) 각 권 소제목을 메인 문서 목차와 전부 대조하여 누락 없음 확인.
  - 卷第五 말미의 발문(跋)에 실린 1512년 간행 정보("皇明正德壬申季冬。府尹...李繼福謹跋")를 확인 — 표준 소스가 실제로 中宗壬申本(규장각본) 스캔에 기반함을 재확인.
  - [2차 작업 시 확정]: 中宗壬申本(규장각본) 원본 이미지 또는 국사편찬위원회 DB와의 자구(字句) 단위 대조는 미실시. 조목(篇/조) 단위 분할 시 정확한 경계 재확인 필요.
- 저작권 상태: 퍼블릭 도메인
- 정규화 내역:
  - CRLF → LF 정규화.
  - 위키 UI/메타데이터 제거: `{{Header2|...}}` 항행 템플릿, `<div style="margin-left:5em">...</div>` 래퍼, ProofreadPage `<pages index="..." from=".." to=".." />` 트랜스클루전 태그, `<noinclude>...</noinclude>` 교정상태 메타데이터, `{{nop}}` no-op 템플릿, 메인 문서의 목차·카테고리·PD 안내 템플릿(권별 파일에는 애초 포함 안 됨) 제거.
  - 절 표제 정규화: `== 제목 ==` / `=== 제목 ===` / `==== 제목 ====` 위키 문법과 `::'''제목'''` 굵게+들여쓰기 문법을 제거하고 제목 텍스트만 별도 줄로 유지(예: "興法第三", "順道肇麗", "居登王" 등 — 편/조목 표제는 원문 구조의 일부로 보존).
  - `<poem>...</poem>` 태그 제거, 내부 향가·찬시(讚詩) 텍스트는 줄바꿈 그대로 보존.
  - `<br/>` 줄바꿈 태그를 실제 줄바꿈으로 치환(표제와 본문이 페이지 경계에서 한 줄에 붙어 있던 경우).
  - `{{*|텍스트}}` 소자쌍행 주석 템플릿(원문의 세주/이표기 주석)을 실제 렌더링과 동일하게 낫표 형태 전각괄호 〈 〉로 치환(원문 괄호 （）와는 구분되는 위키 자체 템플릿이었음).
  - `{{!|문자|자형분해description}}` 희귀자(구성자/IDS) 템플릿을 실제 표시되는 첫 번째 인자(해당 유니코드 문자, 예: 처용가의 𬚵)만 남기고 자형 분해 설명(tooltip)은 제거.
  - `<ref>...</ref>` 판본 이문(異文) 각주(예: 王曆 표 안의 "原本-{「}-...-{」}-" 식 원본 결각/이체자 주기)는 본문에서 제외 — 위키문헌 편집자가 붙인 현대 교감(校勘) 주석이며 원문 본문 흐름에는 포함되지 않는 요소로 판단(2차 작업에서 필요 시 별도 각주 파일로 분리 검토 가능).
  - 卷第一만 예외적으로 두 자료를 이어붙임: (1) mainspace raw wikitext에 직접 평문으로 들어있던 王曆第一 서술형 텍스트, (2) Page:三國遺事 卷第一 1512年 奎章閣本.pdf/31~103 트랜스클루전 원문(紀異第一). 페이지 경계는 원서 그대로 이어붙이며(빈 줄 삽입하지 않음, 단어 중간 분철도 원문 그대로 — 예: "孝" 한 글자만 있는 줄), 두 자료 사이에만 구분을 위해 빈 줄 하나 삽입.

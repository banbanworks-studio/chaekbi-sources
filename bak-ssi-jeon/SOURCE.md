# 박씨전 원문 출처

- 소스 유형: D형 (한국 고전 — 이본 문제)
- 원저자: 미상 / 원제: 박씨전(朴氏傳), 이명 박씨부인전(朴氏夫人傳)
- 판본(저본): 위키문헌 등재본 (문서ID 37625, ko.wikisource.org)
- 저본 선정 근거:
  - 허브 bash 환경에서 접근 가능한 원문 소스가 위키문헌 등재본이 유일함 (국립한글박물관·규장각·경북대 소장본 등은 이미지/논문 형태로만 존재, 텍스트 커밋 불가)
  - 전문이 옛한글로 서술 — 현대역이 아님을 확인
  - 결말에 필사자의 등서(等書) 관지("이 ᄎᆡᆨ이 ᄒᆞ로 젼역 소일 할만ᄒᆞ기로...")가 붙어 있어 특정 필사본의 전산화로 판단, 위키 자체 편집물이 아님
  - **[계열 추정 — 정밀 대조 아님, 성경 확인 기재]**: 서혜은(2008)의 4계열 분류 기준상 결말에 임경업의 죽음·김자점 처형 서사가 없고 임경업의 활약·한탄만 존재 → 2~3계열 경향으로 추정. 소장처 미상이라 원 계열 확정은 불가능하며, 이 상태로 커밋 진행하기로 확인함.
- 소스 경로 (표준): `https://ko.wikisource.org/w/index.php?title=%EB%B0%95%EC%94%A8%EC%A0%84&action=raw`
- 소스 경로 (예비): 없음 (위키문헌이 유일 접근 가능 소스)
- 판본 검증 마커 (하드랩으로 갈릴 수 있으므로 `tr -d '\r' | tr '\n' ' '`로 편 후 grep):
  1. 서두: `아 동방 션죠ᄃᆡ왕 직위 쵸의 ᄒᆞᆫ양 셩ᄂᆡ의 ᄒᆞᆫ ᄌᆡ상이 잇스되`
  2. 결말 등서 관지: `이 ᄎᆡᆨ이 ᄒᆞ로 젼역 소일 할만ᄒᆞ기로 되지 못ᄒᆞᆫ 글시로 등셔 ᄒᆞ엿ᄉᆞ오니`
- 저작권 상태: 위키문헌 `PD-old-100` 분류. 옛한글 필사본 원문 그대로의 전산화이며 현대어 교정본이 아니므로 편집저작권 이슈 없음. 원작 자체도 사후 70년 훨씬 경과(숙종조 추정작).
- 정규화 내역:
  - 위키텍스트 템플릿 제거: `{{옛한글 알림}}`, `{{머리말...}}`, `{{위키백과|박씨전}}`, `{{옛한글/시작}}`~`{{옛한글/끝}}` 태그 자체, `{{PD-old-100}}`, `== 바깥고리 ==` 절, `[[분류:고전소설]]` — 본문(옛한글 시작~끝 사이 텍스트)만 남김
  - CRLF → LF, UTF-8, BOM 없음
  - 문장·철자·구두점·문단 배치 무수정, 하드랩 유지
- 파일 단위: 단일 파일 (원문에 장/회 구분 없는 연속 서사) — `bak-ssi-jeon.txt`
- 비고: 박씨전은 100~140여 종 이본이 현전하는 작품. 이 저본은 소장처 미상 상태로 커밋되므로, 추후 정밀 계열 대조가 필요해지면(예: 타 이본과의 비교 각색 작업 시) 백필 대상으로 재확인할 것.

# 전운치전 원문 출처

- 원저자: 미상 — 주인공 "전우치(田禹治)"는 실존 인물(조선 중종대 방외인)로 알려져 있으나, 소설 「전우치전」의 작자는 전하지 않음.
- 원제: 전운치전 (다른 표기: 뎐운치젼) — 위키문헌 「전우치전」 문서(판본 목록 페이지, pageid 33300)에서 이 작품은 "전우치전(전운치전)"으로 병기되며, 이 저본 자체의 머리말 표기는 "전운치전/뎐운치젼"임.
- 판본: **경판(京板) 37장본** (확정) — 1847년(정미년) 간행, 간기 "丁未仲春由谷新刊"
  - 저본 선정 근거: 위키문헌 「전우치전」 판본 목록에 "경판본 중 최고본"으로 명시된 판본이며, 현재 위키문헌에 전문이 전사되어 있는 유일한 판본임(하단 비고 참조).
- 소스 경로 (표준): `https://ko.wikisource.org/w/index.php?title=전운치전 (경판 37장본)&action=raw` (문서명: "전운치전 (경판 37장본)", pageid 33297, ns0)
- 소스 경로 (예비): `https://ko.wikisource.org/wiki/전운치전_(경판_37장본)`
- 판본 검증 마커: 검증 완료 (2026-07-07) — 정규화된 `ch-01.txt`에서 아래 2개 마커 모두 검출 확인.
  - 마커 1: `丁未仲春由谷新刊` (간기, 1847년 정미년 간행 확정 — 본문 최말미, `{{옛한글/끝}}` 직전)
  - 마커 2: `별호를 구십ᄌᆡ라` (전우치의 별호 "구십재" — 이 판본 고유 표현, 출생담 대목)
- 저작권 상태: 퍼블릭 도메인 — 원작은 저자 미상의 조선시대 고전소설로 저작권 보호기간 만료. 위키문헌 수록본은 `{{PD-old-100}}` 표시. 위키문헌 수록본 자체가 옛한글 표기 그대로의 원문 전사본이며 현대어 번역/현대역이 아님을 확인함(별도의 "/현대어" 하위 페이지 없음).
- 정규화 내역: 위키문헌 `https://ko.wikisource.org/w/index.php?title=전운치전 (경판 37장본)&action=raw`에서 원시 위키텍스트를 curl로 직접 취득(요약 모델 경유 없음). 본문은 `{{옛한글/시작}}`과 `{{옛한글/끝}}` 사이(원본 14~1284번째 줄) 전체를 그대로 추출. 제거한 것은 이 구간 앞뒤의 위키 UI/메타 요소뿐: `{{머리말 ...}}`, `{{옛한글 알림}}`, `{{옛한글/시작}}`, `{{옛한글/끝}}`, `== 라이선스 ==`, `{{PD-old-100}}`, `{{꼬리말}}`, `[[분류:...]]` 4건. 추출 구간 내부에 위키 마크업(`{{`, `}}`, `[[`, `]]`, `<ref>` 등)이 남아있는지 grep으로 전수 검사하여 없음을 확인 — 본문 텍스트(옛한글 표기, 띄어쓰기, 구두점, 문단 구분용 빈 줄, 원본 스캔의 판독 불가 글자를 나타내는 `◯` 표기 포함)는 원문 그대로 한 글자도 수정하지 않음. 원본에 CRLF 없음을 확인(이미 LF), 저장 파일도 LF·UTF-8(BOM 없음)로 저장. 간기 "丁未仲春由谷新刊"은 원문 요소이므로 본문에 그대로 유지.
- 파일 단위: `ch-01.txt` 단일 파일로 전체 본문 수록(1,271줄, 96,425자, 공백 제외 88,309자). 회차 구분이 없는 완결 단일 서사이므로 `ch-XX.txt` 분할 없음 — 홍길동전(`honggildong/ch-01.txt`) 선례와 동일한 방식.
- 비고 (이본 문제 등 저본 선정 근거): 위키문헌 「전우치전」 판본 목록 문서(pageid 33300)에는 경판본 3종(37/22/17장본), 필사본 7종(일사본, 김동욱본, 사재동본, 박순호본, 나손본 등), 활자본 10종(신문관본 외)이 이본으로 등재되어 있으나, API로 개별 확인한 결과 이 중 위키문헌에 실제 전사·수록되어 있는 것은 경판 37장본이 유일하며 나머지는 모두 미작성(red link) 상태임. 따라서 이본 대조는 현재 불가능하며, 추후 다른 이본이 위키문헌에 추가되면 재검토 필요.

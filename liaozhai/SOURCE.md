- 판본 검증 마커:
  - 메인 인덱스 페이지({{header}} 안내문)에 "全書共496篇" 명시 확인.
  - 第01卷 첫 이야기 제목 "考城隍" 및 첫 문장 "予姊夫之祖，宋公諱燾…" 실제 확인.
  - 第05卷 첫 이야기 "陽武侯", 第12卷 첫 이야기 "二班" 및 말미 "附錄"(蟄蛇/晉人/龍/愛才 4편 수록) 실제 확인.
  - 第01卷·第03卷 권말에 "<卷一完>"·"<卷三完>" 표기 실제 확인(다른 권은 별도 권말 표기 없이 다음 이야기로 이어짐 — wikisource 편집 상태의 특성으로 판단, 원문 훼손 아님).
  - 권별 이야기(== 제목 ==) 표제 총 493개(레벨2) + 第12卷 附錄 하위 4편(레벨3: 蟄蛇/晉人/龍/愛才) = 496편으로 집계되어, 메인 인덱스의 "全書共496篇"과 정확히 일치 확인.
  - 12권 전체 HTTP 200 정상 수신 확인(2026-07-07 fetch).
  - 개별 편 단위 문자 대조(주설재초본 영인본 등과의 자구 대조)는 [2차 작업 시 확정] — 이번 1차 작업 범위 밖.
- 저작권 상태: 퍼블릭 도메인
- 정규화 내역:
  - 12권 각 하위 페이지를 `?action=raw`로 원본 위키텍스트 그대로 확보(요약·의역·창작 없음, 렌더링 UI 개입 없이 raw fetch).
  - 위키 UI/네비게이션 요소 제거: `{{header ... }}` 이전/다음 권 링크 틀, `<noinclude>...</noinclude>`(페이지에 렌더링되지 않는 편집자 숨은 주석, 예: 영역본 링크), `== 註釋 ==` 절 + `{{reflist}}` 각주 컨테이너.
  - 부가 주석 제거: `<ref>...</ref>`(위키 편집자가 넣은 현대 어휘 뜻풀이 각주 — 포송령 원저·주설재초본 자체 내용이 아니므로 제거).
  - 후대 평비 삽입 제거: `{{*|...}}` 템플릿으로 표시된 何評(何守奇評)·但評(但明倫評) 등 — 何守奇·但明倫은 주설재초본(1751년 抄) 성립 이후 활동한 평자로, 이 저본 자체 구성 요소가 아닌 위키 편집자의 별도 부가이므로 제거.
  - 서식 태그 제거(내용 문자는 보존): `<u>…</u>`(고유명사 밑줄), `{{ProperNoun|X}}`/`{{專|X}}` → X, `-{X}-`(간체/번체 변환 힌트 래퍼) → X.
  - 이체자 템플릿 정리: `{{另|A|B}}`(이체자 표시) → A(위키 기본 표시자 채택); `{{僻字|A|B}}` → A(실제 벽자 문자).
  - 비유니코드 특수자 처리: 第08卷 "小棺" 편에 유니코드 미수록 부호 2자(위키 전용 커스텀 글꼴에만 종속된 Private-Use-Area 코드포인트, `{{PUA|}}` 템플릿)가 있어 임의 추정/창작하지 않고 "□"(U+25A1, 판독 불능 표시 관례)로 명시 대체 — 해당 편 채택 시 2차 작업에서 별본 대조로 확정 필요.
  - 헤딩 정리: 위키 `==제목==`/`===제목===` 구분 없이 "=" 기호만 제거하고 표제 문자 자체는 무수정 상태로 독립된 한 줄로 유지(이야기/편 경계 표시로 2차 분할 작업에 활용 가능).
  - 개행: CRLF → LF, 3개 이상 연속 개행은 빈 줄 1개로 정리. 원문 단락 들여쓰기(전각공백 "　　")는 원형 그대로 보존.

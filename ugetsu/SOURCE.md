# 우게츠 이야기 원문 출처

- 원저자: 上田秋成 (우에다 아키나리, Ueda Akinari)
- 원제: 雨月物語 (우게츠 모노가타리)
- 판본: 안에이(安永) 5년(1776) 노바이도(野梅堂) 간행 초판 계열 — 청공문고(青空文庫) "校注 雨月物語" (작품ID 60608, 신자구가나) 저본
  - 주의: 청공문고에는 현대어역판(現代語訳 雨月物語, 작품ID 60609, 역자 鵜月洋)도 별도로 존재. 원문 그대로가 필요하면 60608 채택, 60609는 현대어 번역이므로 배제.
- 소스 경로 (표준): `https://www.aozora.gr.jp/cards/001271/files/60608_77891.html` (본문 XHTML, 재확인 및 확정 완료)
  - 실제 저장 작업에는 위 XHTML과 동일 내용을 담은 청공문고 공식 플레인텍스트판 `https://www.aozora.gr.jp/cards/001271/files/60608_ruby_77892.zip` (압축 해제 시 `03kochu_ugetsu_monogatari.txt`)을 사용. XHTML의 `<ruby>`/`<span class="notes">` 마크업과 텍스트판의 `《》｜`/`［＃…］` 표기가 동일 원문을 다르게 렌더링한 것임을 대조 확인(예: 序 첫 문장, 白峯·蛇性の婬·貧福論 도입부 등 다수 구간)했으므로, 무수정 표기가 이미 확립되어 있는 텍스트판을 채택. 두 파일 모두 청공문고 공식 배포물이며 동일 저본(底本)에서 파생됨.
- 소스 경로 (예비): 미확보 상태로 보류 — 표준 소스(아오조라 문고 공식 배포물, XHTML/텍스트판 상호 대조 완료)의 신뢰도가 이미 충분하다고 판단. 필요 시 国立国会図書館デジタルコレクション(NDL Digital Collection)에서 安永 5년(1776) 노바이도 초판 원본 스캔을 대조용으로 확보 예정이나, 착수 시점 이전에는 확보하지 않음.
- 판본 검증 마커:
  - 9편 표제 전부 원문에서 실제 확인 완료 (각 편 앞 ［＃３字下げ］［＃小見出し］...［＃小見出し終わり］ 블록으로 구분): 白峯, 菊花の約, 浅茅が宿, 夢応の鯉魚, 仏法僧, 吉備津の釜, 蛇性の婬, 青頭巾, 貧福論 — 순서·표기(예: 白峯은 峰이 아닌 峯자 사용) 그대로 확인.
  - 5권(巻之一 ～ 巻之五) 구성과 편 배분 확인: 巻之一(白峯·菊花の約), 巻之二(浅茅が宿·夢応の鯉魚), 巻之三(仏法僧·吉備津の釜), 巻之四(蛇性の婬), 巻之五(青頭巾·貧福論).
  - 上田秋成 자서(雨月物語序, 剪枝畸人 명의 한문 서문) 존재 및 본문 포함 확인.
  - 권말 서지: 底本 「改訂　雨月物語　現代語訳付き」角川文庫、KADOKAWA(2006년 초판/2020년 26판), 底本の親本 「雨月物語」野梅堂(1776, 安永5) 확인 — 기존 SOURCE.md 판본 기재와 일치.
  - 확인 결과 이 파일에는 별도의 凡例·解説 섹션이 존재하지 않음(0건, grep 확인) — 아래 정규화 내역 참고.
  - [2차 작업 시 확정]: 9편 각각의 세부 이체자·판본 대조(구자구가나 원본과의 정밀 대조), 삽화(挿絵) 처리 여부.
- 저작권 상태: 퍼블릭 도메인 (저자 몰년 1809년, 일본 저작권법 기준 만료)
- 정규화 내역:
  - 원본은 Shift_JIS 인코딩 → UTF-8로 변환(문자 손실·치환 없음, iconv 변환 로그 확인).
  - CRLF → LF 정규화 완료.
  - 루비(ruby)·편집자주 표기는 청공문고 표준 텍스트판 표기를 그대로 유지 — 제거·가공하지 않음: 루비는 `한자《よみ》` (모호할 때 시작 경계 표시 `｜`), 입력자주(외자 설명·행right小書き 각주 번호·개행/들여쓰기 지정 등)는 `［＃…］` 형식 그대로 보존.
  - 사이트 UI 전용 요소만 제외: (1) 파일 최상단 표기 범례 설명 블록("【テキスト中に現れる記号について】" 및 구분선, 원문 아님·아오조라 사이트 안내문), (2) 파일 최말미 서지사항/입력자·교정자 크레딧/청공문고 안내문("底本：" 이하 전체) — 이는 편집자·아오조라 봉사자에 의한 부가 정보이며 上田秋成의 원문이 아니므로 제외. 이 판본에는 독립된 凡例·解説 섹션 자체가 존재하지 않아(grep 확인, 0건) 그에 대한 별도 판단은 불필요했음.
  - 보존한 본문 범위: "［＃ページの左右中央］"(속표지 정렬 지시, 大見出し "校注　雨月物語" 직전)부터 貧福論 각주 블록 끝("［＃ここで字下げ終わり］")까지 — 序文과 5권 9편 전체 + 각 편 부속 각주(校注) 포함.
  - 현대어역·요약·의역 없음. 신자구가나 표기, 이체자 혼용(底本 그대로라고 명시된 것 포함) 등은 저본 그대로 유지.
- 파일 단위: 1차 작업은 9편 전체를 단일 파일 `wonmun.txt`로 저장(151,746자, 2,018행, UTF-8/LF). 편 경계 자체는 원문 표제 마커로 명확히 구분되나, 序文의 귀속(巻之一 앞에 위치, 특정 편에 속하지 않음)과 각 편 내부 단락 재구성 등 세부 분할 방침이 아직 미정이므로 `tale-001.txt`~`tale-009.txt` 분할은 2차 작업으로 이연. 분할 시 경계 라인(각 편 표제 위치)은 이미 특정 완료: 白峯 L41, 菊花の約 L245, 浅茅が宿 L417, 夢応の鯉魚 L630, 仏法僧 L741, 吉備津の釜 L957, 蛇性の婬 L1173, 青頭巾 L1622, 貧福論 L1816 (wonmun.txt 기준 행 번호).
- 비고 (이본 문제 등 저본 선정 근거, 해당 시): 청공문고 판은 신자구가나(新字旧仮名) 표기 — 원문 고유 표기(구자구가나)와 다를 수 있어 "정규화" 범위에 표기 현대화가 포함되는지 확인 필요.
